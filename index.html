SheepJumpAI

To reset the game with 10 sheep: refresh your browser.

How it works (will change soon):

Currently 10 sheep, each with a neural network attach to them, will spawn and a sheep is eliminated when it hits a fence.
If there are 0 sheep left, the game is over. When there are 2 sheep left, a new generation of sheep (10 sheep) are created.

Created by Emily Su using p5.js, Toy Neural Network Library, HTML, CSS, and JavaScript.
